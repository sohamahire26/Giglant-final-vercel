# Giglant — Professional Freelancer Workflow Tools

Giglant is a suite of professional, browser-based tools designed for video editors, graphic designers, and digital freelancers. It streamlines the "messy" parts of the creative process: file management, client feedback, and delivery.

## Core Features

- **Project Workspaces**: Centralized hubs for managing client projects.
- **Magic Links**: Shareable review pages for clients (no login required).
- **Timestamped Feedback**: Frame-accurate comments for video and audio files.
- **File Renamer**: Batch rename messy files with professional formatting and numbering.
- **Revision Checklist**: Automatically convert client comments into an actionable task list.
- **Delivery Assistant**: Generate professional hand-off messages and payment requests.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS.
- **UI Components**: shadcn/ui (Radix UI), Lucide React, Framer Motion.
- **Backend**: Supabase (Auth, Database, Edge Functions).
- **State Management**: TanStack Query (React Query).

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your Supabase environment variables.
4. Start the development server: `npm run dev`.
